LAMMPS Template

Configured for LAMMPS molecular dynamics simulations. Supports both CPU and GPU execution modes. For jobs >10 nodes, automatically suggests high-bandwidth partition (hbw) for optimal performance.

CPU Example: srun --mpi=pmi2 lmp -in input.in
GPU Example: srun --mpi=pmi2 lmp -sf gpu -pk gpu 2 -in input.in
ANSYS Template

Setup for ANSYS Fluent and Mechanical simulations. Includes HPC Pack license optimization and proper MPI configuration. Supports both CPU and GPU acceleration modes.

Fluent: fluent 3ddp -g -t$SLURM_NPROCS -mpi=intel -cnf=nodelist -i journal.jou
Mechanical: ansys251 -dis -mpi intelmpi2018 -machines $machines -i input.dat
COMSOL Template

Optimized for COMSOL Multiphysics finite element analysis. Supports single-node multithreaded and multi-node MPI+OpenMP hybrid execution. GPU acceleration available for specific physics modules.

Single-node: comsol batch -np 104 -inputfile input.mph -outputfile output
Multi-node: comsol batch -mpibootstrap slurm -inputfile input.mph -outputfile output
